# ブランド設定

## 説明


ブランドにはデフォルトのロゴファイルがあり、デフォルトのロゴは PNG ファイルです。 ユーザーは、ブランドロゴでサポートされているファイルの種類を尋ねることができますか？


## 解像度


サポートされる形式は、.jpeg、.jpg、.gif、.png および.tif です。  また、ブランディングを変更するには、管理者の上に配信品質製品プロファイルが必要です。
