# Adobe Campaign Standard:コレクション内のコレクションを JSON ペイロード内で使用できる

## 説明

ACS がコレクション内のコレクションで JSON ペイロードを受け入れることが可能かどうかを尋ねられました。

## 解像度

現在、ACS ではこの機能はサポートされていませんが、R&amp;D によるこの機能の将来のリリース向けのレビューのための機能強化として、CAMP-47184が開かれています。
