# アセットのローカル参照を JSON 形式で書き出す方法

## 説明


アセットのローカル参照は、プロパティの「参照」タブに表示されます。 （Assets コンソールでアセットを選択し、ツールバーの「プロパティ」を選択して、「参照」タブを選択します）。 AEM以外の情報を使用する場合は、JSON 形式で書き出すことができます。

<b>環境</b>

AEMas a Cloud Service、AEM 6.5


## 解像度


次の URL「1」にアクセスすると、ローカル参照が JSON 形式で返されます。 応答の「path」値は、ターゲットアセットを参照するコンテンツパスです。

「1」 http://localhost:4502/bin/wcm/references.json?path=pathをターゲットアセットに

<b>使用例：</b>
http://localhost:4502/bin/wcm/references.json?path=/content/dam/wknd/en/activities/hiking/equipment_6.jpg
