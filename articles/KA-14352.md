# TLS v1.0 および v1.1 のサポートを削除すると、これらの TLS バージョンを使用して MX サーバーにメールを送信する際に影響はありますか？

## 説明


TLS v1.0 および v1.1 のサポートを削除すると、これらの TLS バージョンを使用して MX サーバーにメールを送信する際に影響はありますか？


## 解決策


いいえ。これらの TLS バージョンは、インスタンスへの HTTPS 接続（コンソール接続や API 呼び出しなど）に関係しています。 SMTP over TLS 設定（メール送信）は変更されません。
