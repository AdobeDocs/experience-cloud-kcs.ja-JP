# TLS v1.0 および v1.1 のサポートを削除すると、これらの TLS バージョンを使用して MX サーバーに E メールを送信する際に影響を受けますか？

## 説明


TLS v1.0 および v1.1 のサポートを削除すると、これらの TLS バージョンを使用して MX サーバーに E メールを送信する際に影響を受けますか？


## 解像度


おこなわれる変更は、インスタンスへの HTTPS 接続（コンソール接続や API 呼び出しなど）に関連するものです。 SMTP over TLS 設定（E メール送信）に変更はありません。
