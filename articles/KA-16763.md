# 変数 c_evar1 のすべての値を取り込む特性を作成する

## 説明

<b>環境</b>
Audience Manager
<b>問題/症状</b>
Adobe Audience Manager(AAM) ですべての受信値を取得する特性を作成する方法を教えてください。


## 解像度


これは、 *正規表現*.

条件： `(c_evar1 matchesregex "*.*$")` は、AAMに入力されるすべての値をキャプチャする特性になります。 これは次のように機能します。 *c_evar1 が存在する場合*.
