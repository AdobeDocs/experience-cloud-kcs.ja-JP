# 購入 ID の購入イベントがレポートに表示されない

## 説明

問題/症状
<br>購入 ID を持つ購入イベントはレポートに表示されません


## 解決策


解決方法購入 ID がすべての訪問者に対して既にレポートスイートで収集されている値と一致する場合、イメージリクエストは重複購入と見なされます。  購入イベントを含むすべてのコンバージョン変数は、レポートに表示されません。

現在、購入 ID（重複購入）はデータ保持期間を過ぎず、永久に存在します。
