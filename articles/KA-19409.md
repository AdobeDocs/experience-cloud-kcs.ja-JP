# [!UICONTROL ファイルコレクター] ACS での代替アクティビティ

## 説明

この [!UICONTROL ファイルコレクター] ACS のアクティビティは、次のアクティビティを使用して構築でき、多かれ少なかれ同じ機能が得られます

## 解決策


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)



この [!UICONTROL ファイルコレクター] は、ファイル名パターンに基づいて複数のファイルを外部サーバーからダウンロードできるようにします。 各ファイルが [!UICONTROL ファイルを読み込み] アクティビティに含まれ、更新されたデータを受け取るアクティビティ。

ファイル転送では、 [!UICONTROL 履歴化設定]知っているように [!UICONTROL ファイルコレクター].
