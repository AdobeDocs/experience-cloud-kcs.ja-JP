# status.adobe.comへの購読を管理

## 説明 {#description}

<b>環境</b>
- Experience Cloud



<b>問題/症状</b>
status.adobe.comの通知を購読または購読を管理する手順です。


## 解決策 {#resolution}


1. status.adobe.comに移動します。
2. Adobe IDを使用してログイン
3. 「購読を管理」をクリックします。
4. 「購読を作成」をクリックします。
5. 「Experience Cloud」の下で、購読用のソリューションとサブコンポーネントを選択します — 続行
6. 地域とイベントの種類の選択 — 続行
7. 購読を確認し、「完了」をクリックします。
