# [ACS] レポートを非表示にするユーザーグループ

## 説明


顧客は、一部のユーザーのすべてのレポートを UI から非表示にする必要があります。

これは ACS で実行できますか？ ドキュメントを確認しましたが、それをおこなう方法が見つかりませんでした。


## 解像度


標準ユーザーの場合、すべてまたは何も実行されません。 したがって、標準ユーザーの ACS インスタンスでは、この種の桜の選択はできません。
