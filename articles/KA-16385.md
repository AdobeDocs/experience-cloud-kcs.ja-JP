# Workspace の上昇率および内訳レポートの信頼性の場合は該当なし

## 説明


該当なしの理由は、 [!DNL Target] Analysis Workspaceの A4T レポートのエクスペリエンス。


## 解像度


上昇率と信頼性は、 [!UICONTROL ターゲット] エクスペリエンスディメンション。 分類は現在サポートされていません。 セグメント化はレポートエンジンの別のレベルで発生するので、回避策として使用できます。
