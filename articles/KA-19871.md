# パストラバーサルは、シークレットへのアクセスにつながります

## 説明 {#description}

<b>環境</b>
Experience Manager6.5


<b>問題/症状</b>
特定の文字を使用してパストラバーサルを取得し、Sling Framework を実装するアプリケーションでのセキュリティメカニズムをバイパスできます。


## 解決策 {#resolution}

Path traversals に取り組むには、AEM v6.5 service pack 11 を適用します。
