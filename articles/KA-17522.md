# Adobe I/Oの設定方法：認証 — ステップバイステップ

## 説明

<br>このドキュメントでは、実行可能な証明書、キー、トークンの作成方法について説明します。<br><br><br><br><br>目的<br><br><br><br> <br><br>
DMA ソリューションの API 呼び出しを実行するAdobe I/Oベアラートークンを生成するための明確で再現可能な手順を提供する。 [!DNL Target].


## 解像度

<br><br>手順<br><br><br><br> <br><br>
<b>Bearer トークンを作成してAdobe I/OAPI 呼び出しを実行する手順：</b>

1. 秘密鍵と公開証明書を生成する
2. 統合コンソール内でのAdobe I/Oの作成
3. JWT トークンを生成
4. アクセスベアラートークンの Exchange JWT トークン
5. API 呼び出しを実行するためのユーザーアクセスベアラートークン


- <b>次のドキュメントに従って、秘密鍵と公開証明書を生成します。</b> https://www.adobe.io/apis/cloudplatform/console/authentication/createcert.html


   Macでは、次のコマンドがターミナルに組み込まれています。 

   コンピューターでは、をダウンロードする必要があります [!DNL Cygwin] （または個人の好みの他のツール）を使用して、コマンドラインから実行します。 次に、をダウンロードしてインストールする手順を示します [!DNL Cygwin]:

   1. https://cygwin.com/install.htmlを参照します。
   2. ダウンロードして実行 `setup-x86_64.exe`

   <b>*注意</b>:  ホームディレクトリは次のとおりです。 `C:\cygwin64\home\WINDOWSUSER`

   インストール段階で、追加のパッケージを検索してインストールできます。 &quot;`curl`&quot;および&quot;`ssh.`&quot;*


   次に、実行するコマンドを示します。
   <br> <br> <br>

   | `$ openssl req -nodes -text -x509 -newkey rsa:2048 -keyout secret.pem -out certificate.pem -days 356` |
   | --- |

<br> 

    &lt;b>注意&lt;/b>:  証明書を生成するために入力が必要な質問がいくつか表示されます。以下のスクリーンショットを参照してください。
![rtaImage](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_0/step_par/image/rtaImage.png "rtaImage")<br> 

    ファイルを作成したら、次のコマンドを使用して「secret.key」を「secret.pem」に変換します。
<br> <br> <br>

    | &#39;$ openssl pkcs8 -topk8 -inform PEM -outform DER -&#39; &#39;in&#39; &#39;secret.pem -nocrypt secret.key&#39; |
