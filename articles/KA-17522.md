# Adobe I/O の設定方法：認証 - ステップバイステップ

## 説明

<br>このドキュメントでは、証明書、キーおよびトークンを作成して実行できるようにする方法を説明します。<br><br><br><br><br>目的<br><br><br><br> <br><br>
[!DNL Target] のような DMA ソリューションに対する API 呼び出しを実行するための Adobe I/O ベアラートークンを生成する、明確で、再現可能な手順を提示します。


## 解決策

<br><br>手順<br><br><br><br> <br><br>
<b>Adobe I/O API 呼び出しを実行するためのベアラートークンを作成する手順：</b>

1. 秘密鍵および公開証明書を生成します
2. Adobe I/O コンソール内に統合を作成します
3. JWT トークンを生成します
4. JWT トークンをアクセスベアラートークンと交換します
5. ユーザーアクセスベアラートークンで API 呼び出しを実行します


- <b>次のドキュメントに従って、秘密鍵および公開証明書を生成します。</b>https://www.adobe.io/apis/cloudplatform/console/authentication/createcert.html


   Mac では、次のコマンドがターミナルに組み込まれています。 

   コンピューター上で、[!DNL Cygwin]（または他の好みのツール）をダウンロードして、コマンドラインから実行する必要があります。次に、[!DNL Cygwin] をダウンロードしてインストールする手順を示します。

   1. https://cygwin.com/install.html を参照します
   2. `setup-x86_64.exe` をダウンロードして実行します

   <b>*メモ</b>：ホームディレクトリは、`C:\cygwin64\home\WINDOWSUSER` です。

   インストール段階で、追加パッケージを検索してインストールできます。「`curl`」および「`ssh.`」* に関連するすべてをインストールすることをお勧めします。


   次に、実行するコマンドを示します。
   <br> <br> <br>

   | `$ openssl req -nodes -text -x509 -newkey rsa:2048 -keyout secret.pem -out certificate.pem -days 356` |
   | --- |

<br> 

    &lt;b>メモ&lt;/b>：証明書を生成するために、入力する必要のあるいくつかの質問をされます。次のスクリーンショットを参照してください。
![rtaImage](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_0/step_par/image/rtaImage.png "rtaImage")<br> 

    ファイルを作成したら、次のコマンドで「secret.key」を「secret.pem」に変換します。
<br> <br> <br>

    | `$ openssl pkcs8 -topk8 -inform PEM -outform DER -` `in` `secret.pem  -nocrypt  secret.key` |
    | --- |

<br> 

    次に、ファイルシステム上に作成されるファイルを示します（この時点で、組織に合わせてこれらのファイルをマシン上の別の場所に移動してもかまいません）。

    メモ：
    
    
    ファイルは、ホームディレクトリ「C:\cygwin64\home\WINDOWSUSER」に作成されます。

   - 会社を選択します
   - 「New Integration」ボタンをクリックします。

- 「New integration」を選択して、「Continue」をクリックします。

![rtaimage_5_](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_1/step_par/image_1811594053/rtaimage_5_.png "rtaimage_5_")

![rtaimage_6_](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_1/step_par/image_559705621/rtaimage_6_.png "rtaimage_6_")

![rtaimage_8_](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_1/step_par/image_282868645/rtaimage_8_.png "rtaimage_8_") 
- 処理が完了したら、「continue to integration details」をクリックします。

- <b>JSON web トークン（JWT）を生成する：</b>
統合 UI で、「JWT」タブをクリックして、秘密鍵を貼り付け、「Generate JWT」ボタンをクリックします。- スクリーンショット：generating JWT - input を参照。

   ![rtaimage_11_](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_2/step_par/image/rtaimage_11_.png "rtaimage_11_") 
   - 生成されたら、JWT およびサンプル CURL コマンドが表示されます。
   - 「Generated JWT」の下にある「Copy」アイコンをクリックします。

![rtaimage_12_](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_2/step_par/image_861805817/rtaimage_12_.png "rtaimage_12_")
