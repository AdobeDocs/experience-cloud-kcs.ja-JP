# 類似モデルを保存できません

## 説明 {#description}

<b>環境</b>
Audience Manager


<b>問題/症状</b><br>AAMでルックアライクモデルを使用している場合、保存が拒否されます。<br>

## 解決策 {#resolution}


問題のモデルが Marketplace のデータを使用している場合は、モデルで使用しているデータがモデリングに対して有効になっていることを確認してください。 そうでない場合、類似モデルは保存されません。 問題がない場合は、サポートケースを開きます。
