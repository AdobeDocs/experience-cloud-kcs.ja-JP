# Launch での Web プロパティの削除に関するAdobe

## 説明 {#description}


<b>環境</b>
Adobe Analytics

<b>問題</b>

これらは、次の節にリストされます。


## 解決策 {#resolution}

現在、プライマリ CDN プロバイダーは、 *Akamai*.

これにより、Web サイトからの Launch タグや Analytics（または他のソリューション）サーバー呼び出しの実行に支障が生じることはありません。
明らかな影響がある。
