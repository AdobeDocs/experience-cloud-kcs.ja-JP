# Analytics Workspace のフィルターまたはセグメントでの滞在時間

## 説明

フィルター/セグメントの処理方法 [!UICONTROL 滞在時間] 指標？

## 解像度


レポートを実行するタイミング：

- 選択した日付範囲でデータが収集されます
- 滞在時間の値が計算されます
- セグメントは結果をフィルタリングします


影響の例：滞在時間値はフィルター/セグメントが実行される前に割り当てられるので、訪問のないページ（ディメンション）に対して滞在時間を割り当てることができます。
