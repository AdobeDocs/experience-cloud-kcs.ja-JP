# Analytics データフィードの geo_country 列の「\*\*\*」

## 説明 {#description}

<b>環境</b><br>Analytics<br> <br><b>問題/症状</b><br>理由： `<b>*` 値を `geo_country` 」列が追加されました。



 

## 解決策 {#resolution}

`</b>*` ヒットに `IPv6` テスト用に指定された値。 この場合、 `geo_country` 戻り値 `<b>*` そして `geo_city` 予約済みを返します。
