# Litmus 統合は実稼動からステージングに移行できますか？

## 説明

Litmus 統合は実稼動からステージングに移行できますか？

## 解像度


いいえ。Litmus 統合は、実稼動からステージングに移動することも、ステージング時に移動することもできません。
