# 複数のメッセージの同期検証。

## 説明

HTTP API を介したストリーミング取り込み中に、同期検証が（またはバッチエンドポイントに対して）機能しない
<br> 

## 解像度


複数のメッセージに対して同期検証を使用することはできません。
「ストリーミング複数のメッセージ」には、1 つのパラメーター ({Connection_ID}) しか使用できません。
