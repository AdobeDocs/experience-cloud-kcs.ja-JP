# 複数のメッセージに対して同期検証が機能しない

## 説明 {#description}


<b>環境</b>

Adobe Experience Platform

<b>問題／症状</b>

HTTP API を介したストリーミング取り込み中、に対して同期検証が機能しない `/batch` endpoint.


## 解決策 {#resolution}


複数のメッセージに対して同期検証を使用することはできません。

AEP では、「複数のメッセージのストリーミング」に使用できるパラメーターは 1 つだけです ( `{Connection_ID}`.
