# 今日の日付のData Warehouseリクエストは、現在のデータを返すことはできますか？ |分析

## 説明


<b>環境</b>
Data Warehouse

<b>問題</b>
今日の日付のData Warehouseリクエストは、現在のデータを返すことはできますか？


## 解像度


<b>回答</b>
はい。現在のデータを返すために、同じ日付（今日）の時間別ファイルを配信するように、時間別 Data Warehouse リクエストをスケジュールできます。
