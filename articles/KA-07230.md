# 今日の日付のData Warehouseリクエストは、現在のデータを返すことはできますか？ | [!DNL Analytics]

## 説明


<b>環境</b>
Data Warehouse

<b>問題</b>
今日の日付のData Warehouseリクエストは、現在のデータを返すことはできますか？


## 解決策


<b>回答</b>
はい。現在のデータを返すために、同じ日（今日）に 1 時間ごとのファイルを配信するように、1 時間ごとのData Warehouseリクエストをスケジュールできます。
